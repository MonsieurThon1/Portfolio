Présentation du Projet

Ce projet consistait à exploiter des données provenant d’OpenStreetMap à l’aide du langage Python.

Le projet avait pour objectif de manipuler des données, automatiser leur traitement et générer des pages HTML.

Travaux réalisés

  • Connexion à une API REST
  • Récupération de données OpenStreetMap
  • Manipulation de fichiers JSON
  • Traitement automatisé des données
  • Création automatique de pages HTML
  • Analyse d’informations géographiques

Technologies utilisées

  • Python
  • API REST
  • JSON
  • HTML
  • Visual Studio Code

Déroulement du Projet

  • Création de groupe
  • Recherche de sujet
  • Collecte de données
  • Manipulation et traitement des données
  • Création de pages HTML

Compétences acquises

  • Programmation Python
  • Manipulation de données
  • Automatisation de tâches
  • Utilisation d’API web
  • Traitement de données structurées
  • Analyse de données

Conclusion

Cette SAE m’a permis de développer mes compétences en programmation Python ainsi qu’en traitement et exploitation de données.
