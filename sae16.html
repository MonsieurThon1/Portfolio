Présentation du projet

Le Projet 16 consistait à réaliser un portfolio personnel permettant de présenter les compétences, projets et apprentissages acquis durant le BUT Réseaux & Télécommunications.

Ce portfolio a été développé sous la forme d’un site web responsive regroupant les différents projets réalisés pendant la formation.

Objectifs du projet

  • Présenter les projets réalisés durant le BUT RT
  • Mettre en avant les compétences acquises
  • Créer un site web moderne et responsive
  • Organiser les contenus de manière claire
  • Valoriser le parcours et le projet professionnel

Travaux réalisés

  • Création de plusieurs pages HTML
  • Développement du design en CSS
  • Ajout d’animations et d’effets visuels
  • Organisation des projets par catégories

Technologies utilisées

  • HTML5
  • CSS3
  • Flexbox
  • CSS Grid
  • GitHub
  • GitHub Pages

Déroulement du projet

  • Recherche de contenu et organisation des projets
  • Création de la structure HTML
  • Développement du design CSS
  • Tests et ajustements pour le responsive design
  • Hébergement du site sur GitHub Pages

Compétences acquises

  • Développement web
  • Création d’interfaces utilisateur
  • Conception responsive
  • Organisation d’un projet web
  • Travail collaboratif
  • Utilisation de Git et GitHub

Lien avec les apprentissages critiques

Ce projet mobilise plusieurs compétences du BUT RT, notamment l’utilisation des outils informatiques, le développement d’applications web, la gestion de données ainsi que le travail collaboratif.

Conclusion

Ce portfolio représente l’ensemble des connaissances et compétences acquises durant ma formation en Réseaux & Télécommunications.
