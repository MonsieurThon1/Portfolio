Présentation du Projet

Ce projet avait pour objectif de créer un site web multipage en HTML et CSS respectant les standards du web modernes.

Le projet devait être responsive, accessible et hébergé sur GitHub Pages.

Travaux réalisés

  • Création d’un site web multipage
  • Mise en place d’une charte graphique cohérente
  • Responsive Design pour mobile et tablette
  • Création d’animations CSS
  • Validation W3C HTML/CSS
  • Utilisation de Flexbox et CSS Grid
  • Hébergement du site sur GitHub Pages

Technologies utilisées

  • HTML5
  • CSS3
  • Flexbox
  • CSS Grid
  • Git & GitHub
  • GitHub Pages

Déroulement du Projet

  • Recherche de sujet
  • Création de la structure HTML
  • Création du design CSS
  • Liaison des pages
  • Ajout de contenu(videos, images)
  • Ajout d'animations

    • Compétences acquises

    • Développement front-end
    • Conception responsive
    • Création d’interfaces web
    • Respect des normes d’accessibilité
    • Gestion de versions avec GitHub
    • Organisation d’un projet web

    Conclusion

    Ce projet m’a permis d’améliorer mes compétences en développement web et de découvrir les bonnes pratiques utilisées dans la création de sites modernes.
